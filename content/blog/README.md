# Blog Posts Directory

Add your blog post MDX files here. Each blog post should have frontmatter with:

```yaml
---
title: "Your Post Title"
excerpt: "Brief description"
date: "2024-12-12"
author: "Dr. Rochita Venkataramanan"
category: "case-discussion" | "radiology-insights" | "patient-education"
featured: true/false
image: "/images/scans/example.jpg"
tags: ["CT", "MRI", "Diagnosis"]
---
```

Example filenames:
- coronary-ct-case-analysis.mdx
- preparing-for-mri-scan.mdx
- understanding-ct-enterography.mdx
